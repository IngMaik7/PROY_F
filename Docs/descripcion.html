Tecnologías Utilizadas

"LO que quieras DECIR"

HTML se utiliza para estructurar el contenido de las páginas.
CSS permite el diseño visual y adaptativo para diferentes dispositivos.
JavaScript(JS) se emplea para funcionalidades dinámicas, incluyendo el manejo del mapa interactivo.
Leaflet.js es una biblioteca de código abierto que facilita la visualización de mapas y polígonos en la web.
OpenStreetMap provee los datos cartográficos base para los mapas mostrados en el sitio.

Gracias a estas herramientas, el proyecto puede ofrecer una experiencia visual rica, interactiva y educativa.
